# MERN Job Board Application

A simple and clean Job Board web application built using the **MERN Stack (MongoDB, Express, React, Node.js)**.  

The application allows users to:

- View job listings  
- Filter jobs by category  
- Post new job listings using a form  
- Fetch and store job data through a backend REST API
